#+TITLE: 750words command-line client
#+author: Diego Zamboni

A simple client for posting text to your [[https://750words.com/][750words.com]] account. Can also be used to query your current word count or the current text. Emacs integration is also available.

You can see its full annotated source code in the [[https://github.com/zzamboni/750words-client/blob/main/750words-client.org][750words-client.org]] file.
